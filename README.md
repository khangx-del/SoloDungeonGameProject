Dungeon Labyrinth

A dungeon-crawling adventure built with Unity where you explore a dangerous labyrinth and hallways, fight enemies, and grow stronger with every level.

🎮 Game Overview

In Dungeon Labyrinth, you play as a lone adventurer trapped inside a mysterious maze filled with enemies and powerful bosses. Fight your way through the labyrinth, level up your character, and choose upgrades to survive longer and defeat increasingly difficult foes.

✨ Features
🧭 Labyrinth-style dungeon exploration
⚔️ Combat system with normal enemies
👹 Two challenging boss fights
📈 Level-up system with upgrade choices
🎯 Player progression to make future runs easier
🕹️ Simple and responsive controls
🕹️ Controls
Input	Action
W / A / S / D	Move character
Mouse Movement	Aim
Right Click	Attack
